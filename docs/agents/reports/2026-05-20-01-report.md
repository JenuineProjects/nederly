## 2. Researcher findings — 5-line summary

1. **Top pronunciation traps** (cited): `g/ch`, `ui`, `eu`, `uu`, `ee/oo` flat-hold, `ij/ei`, schwa, `sch`, clusters, fast speech. User's own transcription column confirms the §1A top-5 directly.
2. **Top grammar traps**: V2 word order, subordinate-clause verb-final, `de/het`, diminutives (5 forms), number inversion, modal+main-verb splitting.
3. **Engagement evidence** converges on: 15–30 min daily, microlearning 5–15 min, spaced > massed, audio-first for adults, gamification helps motivation but **streaks specifically trigger loss-aversion** (externally validates the Dyslexia Expert's no-streaks call).
4. **Typical first 5 lessons** of beginner Dutch cover greetings + *zijn/jij/u* → origin/*hoe gaat het* → family + *hebben* + possessives → numbers/days/time → house + food + modals. User's class is slightly behind (phonetics-front-loaded).
5. **Dyslexia + L2**: phonological deficits transfer L1→L2 and affect listening (not just reading); "language spontaneity deficit" proposed for adult dyslexics — production under time pressure is disproportionately hard.

## 8. Lessons for next run

1. **Researcher saves agent runtime.** Downstream agents stopped having to "guess at evidence" — Dutch Expert reordered the trap table to match the cited ranking; Dyslexia Expert cited [S44][S45] for streaks; Phonetics promoted g/ch to #1 on the back of cited evidence. Keep the Researcher in the pipeline.

2. **Manager memory makes the spawn prompts tighter.** Knowing VERSION = 3 → 4, knowing OpenDyslexic was already loaded, knowing the `#6B6B6B` fix was shipped — all of this would have been re-asked otherwise. Memory pays for itself within one run.

3. **Non-words slip through without phonetic re-review.** `noos` (last run) and `hijs` (last run) both survived two prior pipeline runs before Phonetics + Researcher together flagged them. **Rule for future runs:** the Phonetics gate must explicitly verify every minimal pair word against "is this a real A0 standalone Dutch word?" before passing.

4. **Spec citations were strong this run.** Every Designer decision pointed back to a specific section in an upstream doc — this made Gate 2 review trivial. Make this the standard.

5. **Engagement features have to be paired with the no-streaks ban.** It's easy to say "no streaks" and then forget to fill the motivational gap. The Dyslexia Expert's §11 (engagement without streaks) was an important addition — without it, the no-streaks rule reads as deprivation. Future engagement features should always cite Dyslexia Expert §11.

6. **One file flagged as still missing:** `OpenDyslexic-Regular.woff2` has been pending for two runs. Surface this in the user's status update each run until it's resolved.
